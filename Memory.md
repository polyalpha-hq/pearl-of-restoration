## Экосистема — четыре компонента

### 1. Industrial Oil Regeneration System ✅ ГОТОВО К ПРОДАЖЕ
- 3 станции физически существуют
- Производительность: 500 л/час / 4 млн л/год
- Экономия клиента: до 70%
- Окупаемость для клиента: < 12 месяцев
- Себестоимость очистки: < $0.01/литр
- Патент WIPO зарегистрирован

### 2. Metalworking Fluid Concentrate 🔄 В ПРОДАЖАХ
- Производится из восстановленного масла
- Аналог Blasocut 2000 Universal
- Текущие клиенты: предприятия вертолётостроения

### 3. Activated Carbon from Organic Waste 🔄 В РЕАЛИЗАЦИИ
- Сырьё: "agricultural biomass waste" (публично не уточнять источник)
- Продукт: activated carbon (premium grade)
- Первый комплекс: CAPEX ~$7M, контракт на 300 т/год
- **Не упоминать**: мелассу, сахарные заводы, Воронеж

### 4. Waste-to-Energy Complex 📋 КОНЦЕПЦИЯ
- Статус: нет действующего предприятия
- На сайте: позиционировать осторожно, без конкретных обещаний

## Принципы работы с кодом

- Верификация в браузере: использовать `npx serve`, НЕ `python3 -m http.server`
  (python не поддерживает HTTP Range requests — видео не воспроизводятся)
- Пути к медиа: всегда relative (`media/...`), без ведущего слеша и без `https://`
- GitHub Pages: static only, no server-side execution
- Telegram bot: все покупки и реферальные действия — только через бот
